#  Natural Language Processing (NLP) Lab

##  Overview

This repository contains my solution for the **Natural Language Processing (NLP) Lab Assignment**, where I built a text preprocessing pipeline for hotel reviews. The project focuses on cleaning, transforming, and analyzing customer feedback to prepare it for sentiment analysis.

##  Objectives

- Clean and preprocess hotel review text
- Tokenize and normalize text data
- Remove stopwords and apply lemmatization
- Generate n-grams and analyze frequent terms
- Perform sentiment analysis using VADER
- Evaluate the effectiveness of the preprocessing pipeline

## 🛠️ Technologies Used

- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Key NLP Techniques

- Text Cleaning
- Tokenization
- Stopword Removal
- Lemmatization
- N-gram Generation
- Frequency Analysis
- VADER Sentiment Analysis
